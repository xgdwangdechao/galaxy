virtual std::optional<nlohmann::json> galaxy::Serializable::serialize ( )
pure virtualnoexcept

Serialization interface.

You need to fill out a json object with data from the class to be serialized and return it from this object. You will need to call std::make_optional. Cannot throw exceptions.

Returns
Json object containing data to be serialized, as an std::optional, to test if there was any errors creating object.
